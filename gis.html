CRSP Form Generator

For this project, I created a Python pipeline in Jupyter Notebooks that ran through a large database provided by MnDOT. This pipeline generated 200+ informational layouts about intersections, road curves, and road segments and exported them to PDFs. Each layout had 30+ fields that were filled out using a SearchCursor that ran through the database. This pipeline reduced workload and allowed the team to make changes easily and efficiently.

I-94 Removal Route Analysis

For this analysis, I chose to compare the differences in service area for cars before and after the removal of I-94 between downtown Minneapolis and Saint Paul. To achieve this, I created multiple Network Analysis Datasets using the street centerlines of the seven county Twin Cities metro. One dataset had I-94, while the other did not. With these datasets, I was able to randomly sample points around the metro and compare how their service areas were changed. More about this project will be included as it is finalized.

Data Request Tracking Form

This was a Survey123 form I created to enable project managers to request data. This data came in the form of videos of intersections that needed vehicle and/or pedestrian turning movememts counted. This tracking form enabled project managers to select the exact location they wanted counted, input deadlines, and billing information. This form also connected to a web map on ArcGIS Online, this allowed project managers to visualize where data has been collected to avoid redudancy.
